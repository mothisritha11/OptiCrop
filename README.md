# 🌱 OptiCrop - Smart Crop Recommendation System

## 📌 Overview

OptiCrop is a Machine Learning-based crop recommendation system that helps farmers make data-driven decisions. By analyzing environmental and soil parameters such as temperature, humidity, rainfall, and soil nutrients, the system predicts the most suitable crop to cultivate for maximum yield and sustainability.

This project aims to improve agricultural productivity, reduce risks, and promote efficient resource utilization.

---

## 🚀 Features

* 🌾 Predicts best crop based on input parameters
* 📊 Uses Machine Learning algorithms for accurate results
* ⚡ Fast and user-friendly interface
* 🌍 Helps in sustainable farming decisions
* 📈 Improves crop yield and reduces losses
* 🧠 Data-driven agricultural insights

---

## 🧠 Machine Learning Overview

The system uses supervised machine learning techniques trained on agricultural datasets containing:

* Soil nutrients (N, P, K)
* Temperature
* Humidity
* Rainfall
* pH values

Algorithms such as **Decision Trees / Random Forest / Naive Bayes** are commonly used for crop prediction models.

Machine learning helps analyze patterns in environmental conditions and recommends crops that are most suitable for specific regions, improving efficiency and productivity.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/mothisritha11/OptiCrop.git
cd OptiCrop
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Application

```bash
python app.py
```

### 4️⃣ Open in Browser

```
http://127.0.0.1:5000/
```
